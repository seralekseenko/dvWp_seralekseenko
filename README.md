## run in IDEA:
just run :)

## build in host commandline:

$ cd /services/app/

$ mvn package

## run in host commandline:
$ cd /services/app/target

$ java -jar app.jar

## Run in docker container

### Docker composer run:

docker-compose up -d

### Docker composer stop:

docker-compose stop

## Usage

- http://localhost:4567/hello
- http://localhost:4567/1
- http://localhost:4567/trololo
